Required Forms

Last updated: December 14, 2018

The forms are all online this year, so please submit them by January 6, 2019.

Parent Permission Form

Please fill out the Parent Permission Form and observe the following points.

  • Enter the parent’s name and email in the first screen that comes up after clicking the link. This means that even if a student is over 18, the parent needs to put down their information and sign for the student. There is no field for student signature, just parent signature.
  • Everyone should use the new link, and all who used the old link need to resubmit the form again using the new link.
  • Double check that the checkbox is checked off when submitting the form.

Coach's Form

For coaches, the form is still the same, though we’ve removed the previous responses since we have a new form. Please fill it out here.

MIT Alert

Additionally, all attendees are required to be signed up for MIT Alert, the automatic messaging system used by MIT under emergencies. Please have all students, parents, and coaches fill out this form so we can submit the contact information accordingly to MIT.
